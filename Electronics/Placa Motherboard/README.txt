La carpeta 'Placa Motherboard' cuenta con los archivos PCB y esquemático
para ser editados mediante cualquier programa de edición de placas (preferentemente Altium).
Se recomienda no realizar cambios sustanciales en la placa ya que pueden interferir
con su correcto funcionamiento. El archivo editable se provee con la idea de poder reemplazar componentes
que difieran del tamaño original.

El archivo .pdf es el documento listo para imprimir en una placa de cobre simple faz de 15 cm x 10 cm.

Se provee una imagen del PCB y esquemático.

El archivo 'Lista de Materiales' provee una lista de los diversos componentes así como la nomenclatura
asociada a la placa para facilitar su fabricación

-----------------------------------------------------------
The folder 'Placa Motherboard'contains the PCB file and schematic to be edited through any PCB board designer
(Alium Designer recommended). It is adviced not to make sustancial changes in the board for this can interfere
with the device proper behaviour. The editable files purpose is to provide the possibility to edit components 
that differ from the original size.

The .pdf file is the PCB document ready to print in a 15 cm x 10 cm one copper layer PCB board.

The file 'Lista de Materiales' provides a list of the different components as well as their designators
in the PCB board to facilitate its fabrication.
